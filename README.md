Realizo Indicadores Clave de Rendimiento o KPI y segmentaciones de datos de las campañas de marketing (rangos de edad, rangos de renta, 
tipos de productos, de establecimientos) para identificar el perfil de cliente de una empresa a través de Microsoft Power BI.

Microsoft Power BI es un programa que permite realizar métricas (medianas, porcentajes, sumas, restas, multiplicaciones, divisiones, etc), 
y representaciones gráficas de las medidas a través de mapas de geolocalización, segmentacion de datos (como, por ejemplo, rangos de edad,
rangos de renta, año, trimestre, nivel de formación, número de hijos, tipo de producto comprado, etc), gráficos circulares, gráficos de barras, 
gráficos de líneas, tarjetas visuales de los Indicadores claves de rendimiento o KPI (como, por ejemplo, el objetivo de ventas de cada campaña de ofertas, 
el número de ventas reales obtenidas y el porcentaje de aumento o reducción entre el objetivo y el número de ventas reales).
